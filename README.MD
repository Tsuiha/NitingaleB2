# Nitingale B2

![Nitingale B2 Overview](https://raw.githubusercontent.com/Tsuiha/NitingaleB2/main/pic/nitnigale_b2_2.JPG)

「Nitingale B2」は、ゲーミングデバイスとして製作された左手デバイス／右手デバイスで、メカニカルスイッチを使用した自作キーボードです。

---

## 特徴

- **キー配列**  
  [Willow配列](https://coal.hatenablog.com/entry/2020/12/04/231923) や[Project mammoth](https://github.com/illness072/project-mammoth) からインスピレーションを受けてデザインされたキー配列で、無理のない指の可動域による運指をコンセプトとしています。  
  キースイッチ30個 + レバースイッチ1個の構成で、33のキー入力が可能です。

- **用途**  
  ゲーミングやアプリケーション操作を想定しており、多目的な用途に使用できるデバイスです。  
  左手用に加え、右手用もラインナップしています。

- **対応キースイッチ**  
  「Cherry MX」または「KS-33B Gateron Low Profile 3.0」に互換性のあるスイッチが使用できます。  
  キースイッチはMX用ホットスワップソケットにてPCBに接続します。

- **サイドスイッチ**  
  背面に3方向（左、右、押し込み）のレバースイッチを実装しています。  
  プロファイル切り替えやワンタッチで行いたい操作を割り振ることができます。

- **チルト／テントアングル**  
  専用ケースはチルト角6度、テント角3度の傾斜が設けられています。

## 仕様

| 項目 | 内容 |
|------|------|
| 接続 | USB Type-C（有線） |
| 入力数 | キースイッチ30入力／レバースイッチ3入力 |
| ファームウェア | QMK Firmware, [Nitingale B2 codes](https://github.com/Tsuiha/vial-qmk/tree/vial/keyboards/0_tsuiha/nitingale_b2)  |
| 設定ソフト | Vial |

---

## 構成

`Plate` - `PCB` - `Case` の3パーツ構成で、プレートマウント方式です。  
PCBはキースイッチを介してプレートに固定され、プレートはスタンドオフスペーサを介してケースに固定されます。
